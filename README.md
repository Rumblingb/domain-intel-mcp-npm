# @agentpay/domain-intel-mcp

Domain intelligence: DNS, SSL, IP geolocation, and health checks

This npm package wraps the Python MCP server at [Rumblingb/domain-intel-mcp](https://github.com/Rumblingb/domain-intel-mcp).

## Install

```bash
npm install -g @agentpay/domain-intel-mcp
pip install mcp
```


## Usage

```json
{
  "mcpServers": {
    "domain-intel-mcp": {
      "command": "npx",
      "args": ["@agentpay/domain-intel-mcp"]
    }
  }
}
```
